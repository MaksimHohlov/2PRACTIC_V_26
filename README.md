# Инструмент визуализации графа зависимостей

Инструмент для анализа и визуализации зависимостей пакетов Python.

## Этап 1: Минимальный прототип с конфигурацией

### Функциональность

- Загрузка конфигурации из TOML файлов
- Валидация всех параметров конфигурации
- Обработка ошибок для всех параметров
- Вывод конфигурации в формате ключ-значение

### Использование

```bash
python dependency_visualizer.py config.toml
